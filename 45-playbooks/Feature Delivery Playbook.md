# Feature Delivery Playbook

> [!summary] Goal
> Move a bounded task from intent to merge with clear evidence and minimal process drift.

## Steps

1. Clarify the task objective and acceptance criteria.
2. Assign the branch and scope.
3. Retrieve only the necessary repo context.
4. Execute the local code-test-commit loop.
5. Open or update the draft PR.
6. Attach evidence.
7. Run CI and independent review.
8. Merge only when the definition of done is met.
9. Feed follow-up observations back into the backlog.
