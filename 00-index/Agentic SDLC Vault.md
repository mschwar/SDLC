## Vault Map

- Layers: the end-to-end lifecycle in operating order
- Reference: reusable concepts and definitions
- Patterns: optional advanced approaches for scale, autonomy, or complexity
- Variants: repo- and environment-specific adaptations of the core blueprint
- Playbooks: stepwise operating guides for recurring workflows
- Templates: reusable artifacts for repo setup and implementation
- Starter Kits: copyable repo skeletons derived from the worked examples
- Examples: suggested rollout and adoption paths
- Worked example repos: [[50-examples/Solo Python Task Router Example]], [[50-examples/Small Team Release Gateway Example]], [[50-examples/Regulated Change Control Example]], and [[50-examples/Monorepo Delivery Portal Example]]
- Archive: research inputs and historical snapshots
