# Adversarial Verification

> [!abstract] Definition
> Review and testing performed from the perspective of a breaker rather than a builder.

## Forms

- adversarial reviewer prompts
- security-agent checks
- fuzzing
- mutation testing
- using a different model family for review than for implementation
