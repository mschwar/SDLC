# Git Hooks

> [!abstract] Definition
> Scripts Git runs automatically at specific lifecycle moments such as pre-commit, commit-msg, pre-push, post-commit, and post-merge.

## Why They Matter

Hooks turn quality checks from advice into structure.

## Common Uses

- block bad commits
- validate commit messages
- auto-push branches
- create PRs
- remind humans to sync dependencies after merges
