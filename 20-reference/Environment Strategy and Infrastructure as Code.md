# Environment Strategy and Infrastructure as Code

> [!abstract] Definition
> The plan for how development, test, staging, and production environments are structured and how they are created or changed.

## Good Environment Strategy

- environments have clear purpose
- promotion flow is explicit
- infra changes are reviewed like code
- environment drift is minimized
