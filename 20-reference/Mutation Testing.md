# Mutation Testing

> [!abstract] Definition
> Intentionally change code in small ways and check whether the tests fail. Surviving mutations reveal blind spots in the test suite.

## Why It Matters

Coverage tells you what was executed. Mutation testing tells you whether the tests were sharp enough to notice damage.
