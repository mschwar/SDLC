# Documentation Governance

> [!abstract] Definition
> The rules for which notes are canonical, how they are maintained, when they must be updated, and how drift is detected.

## Good Governance Rules

- one canonical note per concern
- explicit archive path for retired docs
- docs update requirements in the definition of done
- periodic review of high-authority notes

## Why It Matters

Agentic systems fail quietly when outdated docs continue to look authoritative.
