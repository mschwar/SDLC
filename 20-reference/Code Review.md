# Code Review

> [!abstract] Definition
> Independent examination of a proposed change before it becomes part of the protected branch.

## What Good Review Checks

- scope alignment
- correctness
- safety
- pattern fit
- evidence quality

## Agent-Specific Rule

Do not let the same agent both author and approve the change.
