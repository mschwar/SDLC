# Branch Protection Rules

> [!abstract] Definition
> Repository-level rules that block unsafe actions on important branches.

## Typical Rules

- require status checks
- require review approval
- block force-push
- require up-to-date base branch

## Why They Matter

These rules stop agents from bypassing process under pressure.
