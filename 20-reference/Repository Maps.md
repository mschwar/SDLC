# Repository Maps

> [!abstract] Definition
> Lightweight indexes that help an agent find the right files, symbols, and boundaries without loading the whole repo into context.

## Examples

- `ctags`
- language-server symbol graphs
- curated file manifests
- ownership maps

## Why They Matter

Repository maps improve retrieval quality and reduce the temptation to compensate with longer, less precise prompts.
