# Feature Flags and Progressive Delivery

> [!abstract] Definition
> Techniques for controlling exposure of new behavior after code is merged, rather than coupling rollout directly to deploy time.

## Why They Matter

Feature flags reduce blast radius, enable partial rollout, and separate shipping code from exposing behavior.
