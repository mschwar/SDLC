# Data Governance and Privacy

> [!abstract] Definition
> The rules for classifying, handling, storing, transmitting, and exposing data within an agent-driven workflow.

## What To Define

- data classes
- allowed tools and models by data class
- retention and logging rules
- redaction and minimization requirements
- approval boundaries for sensitive data access
