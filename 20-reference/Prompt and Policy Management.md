# Prompt and Policy Management

> [!abstract] Definition
> The controlled maintenance of system prompts, role prompts, task instructions, and behavioral policies that shape how agents operate.

## What Good Management Includes

- canonical prompt locations
- versioned prompt changes
- clear policy inheritance
- review for safety-critical prompt changes
- prompt drift detection

## Why It Matters

In mature agent systems, prompts and policies are code. Treating them as throwaway text creates invisible behavior changes.
