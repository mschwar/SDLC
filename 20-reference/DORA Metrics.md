# DORA Metrics

> [!abstract] Definition
> A set of delivery performance metrics centered on lead time, deployment frequency, change failure rate, and recovery time.

## Use In Agentic Context

Translate the spirit of DORA into repo-visible measures:

- check pass rate
- PR cycle time
- rework rate
- recovery speed
