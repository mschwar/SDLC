# Versioning and Release Trains

> [!abstract] Definition
> The rules for how changes are versioned, grouped, and promoted over time.

## Common Choices

- continuous release from `main`
- semantic versioning
- calendar versioning
- scheduled release trains

## Why It Matters

Without explicit versioning rules, delivery speed can outpace clarity about what shipped and when.
