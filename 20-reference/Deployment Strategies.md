# Deployment Strategies

> [!abstract] Definition
> The controlled methods used to expose validated code to runtime environments.

## Common Strategies

- direct deploy
- staged rollout
- canary
- blue-green
- feature-flagged release
- rollback to last known good state

## Selection Criteria

- blast radius
- reversibility
- traffic shape
- operational maturity
