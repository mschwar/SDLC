# CI-CD

> [!abstract] Definition
> Continuous Integration re-runs builds and tests on every change. Continuous Delivery or Deployment moves validated code toward its runtime target.

## Why It Matters

Local checks can be bypassed or run in inconsistent environments. CI verifies the branch in a clean environment. CD turns validated merges into reliable releases.
