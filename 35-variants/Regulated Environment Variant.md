# Regulated Environment Variant

> [!summary] Fit
> Use this when traceability, access control, audit evidence, or compliance review materially change how the repo operates.

## Additions

- stronger approval gates
- tighter identity and permission controls
- durable audit logs
- explicit data handling rules
- formal evidence requirements for release and review

## Risks

- undocumented agent actions
- unclear data boundaries
- compliance work treated as an afterthought
