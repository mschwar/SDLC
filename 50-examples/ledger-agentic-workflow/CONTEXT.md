# Ledger Agentic Workflow Example

This directory contains a snapshot of the agentic workflow, hooks, and operating rules from the `ledger` repository. It serves as a real-world reference for implementing the Agentic SDLC blueprint.

## Contents

- `AGENTS.md`: Canonical workflow, delegation, and PR rules.
- `RUNBOOK.md`: Operating and recovery reference.
- `README.md`: High-level system overview.
- `.githooks/`: Git hooks for linting, commit messages, and automated PR creation.
- `.github/workflows/`: CI workflow for quality checks.
- `templates/`: Pull request and other workflow templates.
